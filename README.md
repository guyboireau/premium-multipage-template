# Template Vitrine Premium (Multi-Pages)

Ce template est conçu pour les entreprises du bâtiment et de la rénovation souhaitant une présence en ligne haut de gamme, inspirée du style de `hauger-renovation.fr`.

## ✨ Caractéristiques

- **Multi-pages** : Accueil, Services, Réalisations (Portfolio), Contact.
- **Design Premium** : Utilisation de typographies élégantes (*Playfair Display* & *Outfit*), palette de couleurs "Navy & Gold", et animations fluides.
- **Performance** : Propulsé par Astro pour une rapidité optimale (zéro JS par défaut, ViewTransitions pour l'effet "App").
- **Responsive** : Parfaitement adapté aux mobiles et tablettes.
- **SEO Ready** : Structure sémantique HTML5, balises meta optimisées.

## 🚀 Structure du Projet

- `src/layouts/Layout.astro` : Le squelette du site (Header, Footer, Styles globaux).
- `src/components/` : Composants réutilisables (Hero, ServiceGrid, Portfolio, etc.).
- `src/pages/` : Les différentes pages du site.

## 🛠 Personnalisation Rapide

1. **Couleurs** : Modifiez les variables CSS dans `:root` dans `src/layouts/Layout.astro`.
2. **Contenu** : Les textes sont facilement modifiables dans chaque page.
3. **Images** : Remplacez les URLs Unsplash par vos propres photos dans les composants.

## 📦 Installation

```bash
npm install
npm run dev
```
